# Interview Prep Skill

AI-powered interview preparation tool for technical interviews.

## Features

- **Question Bank**: Common technical interview questions by category
- **Mock Interview**: Practice mode with timed responses
- **Answer Review**: Get AI feedback on your answers
- **Company Research**: Prepare for specific companies
- **STAR Method**: Guide for behavioral questions

## Commands

```bash
# Start a mock interview
interview-prep start --category javascript --count 5

# Practice specific topic
interview-prep practice --topic system-design

# Review your answers
interview-prep review --file answers.md

# Company-specific prep
interview-prep company google --role frontend

# Behavioral questions
interview-prep behavioral --method star
```

## Categories

- JavaScript / TypeScript
- React / Vue
- Node.js / Python
- System Design
- Database / SQL
- Algorithms / Data Structures
- Behavioral / STAR

Generates practice sessions, answer guides, and progress reports.
