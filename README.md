# OpenClaw Skills

可复用的 OpenClaw 技能包集合。

## 包含技能

- 🌤️ 天气查询 - 获取实时天气信息
- ⏰ 提醒功能 - 定时任务提醒
- 📊 监控系统 - 系统资源监控
- 🔮 星座运势 - 每日星座运势
- 🧠 记忆系统 - 长期记忆管理
- 🖼️ 图片生成 - AI 图片生成

## 安装

```bash
# 克隆仓库
git clone https://github.com/adenzhou1350/openclaw-skills.git

# 查看技能列表
ls skills/
```

## 使用

详见各技能目录下的 SKILL.md 文件。
